# Raytracer

A Ray tracer made in C++, with reflection and refraction . I made this following the awesome tutorial of [Computer Graphics from 'ssloy'](https://github.com/ssloy/tinyraytracer/wiki). The materials you see are:

- White -> Ivory
- Red -> Rubber
- Center -> Glass
- Up Left -> Mirror
